mysql在windows下的安装，mysql官网下载

1.以管理员身份运行cmd,cd mysql解压目录的bin下，执行mysqld –install
2.默认配置文件my-default.ini，可以创建自己的配置文件my.ini
3.启动命令：net start mysql
4.将MySQL的bin路径加到path中
